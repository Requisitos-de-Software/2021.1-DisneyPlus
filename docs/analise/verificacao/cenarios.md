## 1. Metodologia
&emsp;&emsp;Para verificação do artefato de cenários, foi utilizada o método de inspeção, qual o objetivo é encontrar problemas por meio de uma avaliação no documento. Para facilitar a inspeção, foi criado um checklist para verificar se os cenários estão de acordo com o que se espera.
