&emsp;&emsp;Esse é o check-list utilizado para a elaboração das respostas quanto ao rich picture.

    2. Todos os elementos existem?
    3. Os simbolos estão claros?
    4. Possui fronteiras?
    5. Textos estão claros?
    6. As relações fazem sentido?
