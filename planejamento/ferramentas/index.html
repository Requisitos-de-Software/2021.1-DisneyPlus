    Ferramenta Finalidade
    Google Workspace Criação do questionário e planilhas
    Discord Reunião da equipe para elaboração das atividades
    Github Hospedagem do repositório para o projeto
    Microsoft Teams Reunião da equipe para gravar as apresentações
    WhatsApp Utilização de grupo para conumicação entre os integrantes da equipe
    Figma Utilizado para elaboração dos Rich Pictures
